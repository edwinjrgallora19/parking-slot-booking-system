# Parking Slot Booking and Monitoring System

This project is a web-based system that allows users to:
- View available parking slots
- Reserve parking spaces
- Perform QR-based check-in and check-out

## Tech Stack
- Frontend: React
- Backend: .NET API
- Database: MSSQL

## Features
- Reservation system
- QR validation
- Admin dashboard
- Role-based access control (RBAC)
